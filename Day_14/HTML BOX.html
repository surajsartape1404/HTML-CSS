Box Model:

The box model describes how elements are rendered on a webpage, consisting of content, padding, border, and margin.

  • Content : HTML Text, Image
  • Padding : Space between the content and the border. Here we not have a styling, we can apply only diamensions.
  • Border: A border surrounding the padding and content.
  • Margin: Space outside the border, separating this element from other elements. Here we not have a styling, we can apply only dimensions.
